# 🚩 Signal Flags 음성 파일 구조

## 📁 폴더 구조
```
assets/
├── male/           # 남성 음성 파일
│   ├── _up.mp3     # "UP!" 명령
│   ├── _down.mp3   # "DOWN!" 명령  
│   ├── _left.mp3   # "LEFT!" 명령
│   └── _right.mp3  # "RIGHT!" 명령
├── female/         # 여성 음성 파일
│   ├── _up.mp3     # "UP!" 명령
│   ├── _down.mp3   # "DOWN!" 명령
│   ├── _left.mp3   # "LEFT!" 명령
│   └── _right.mp3  # "RIGHT!" 명령
└── README.md       # 이 파일
```

## 🎵 음성 파일 요구사항

### 파일 형식
- **형식**: MP3 (웹 브라우저 호환성)
- **품질**: 128kbps 이상 권장
- **길이**: 1-2초 (짧고 명확하게)
- **볼륨**: 정규화된 레벨 권장

### 음성 내용
- **남성 음성**: 저음, 명확한 발음으로 "UP!", "DOWN!", "LEFT!", "RIGHT!" 
- **여성 음성**: 고음, 명확한 발음으로 "UP!", "DOWN!", "LEFT!", "RIGHT!"
- **언어**: 영어 (게임 규칙상 중요)

### 게임 규칙 연관성
- **여성 음성**: 플레이어가 명령 그대로 수행 (UP → 위로)
- **남성 음성**: 플레이어가 명령 반대로 수행 (UP → 아래로)

## 🔧 파일 준비 방법

### 1. Text-to-Speech 서비스 이용
- Google Cloud Text-to-Speech
- Amazon Polly
- Microsoft Azure Speech
- 무료 대안: Balabolka, eSpeak

### 2. 음성 녹음
- 실제 남성/여성 성우 녹음
- 마이크 녹음 후 편집

### 3. 무료 음성 리소스
- Freesound.org
- Zapsplat (무료 계정)
- BBC Sound Effects Library

## 🎮 테스트 방법

게임에서 오디오 파일을 테스트하려면:

1. 음성 파일을 해당 폴더에 배치
2. 서버 재시작: `npm start`
3. 게임 접속 후 두 명의 센서 연결
4. 게임 시작하여 음성 명령 확인

브라우저 개발자 도구 콘솔에서 오디오 로드 상태를 확인할 수 있습니다.

## 📝 참고사항

- 음성 파일이 없어도 게임은 정상 작동 (시각적 명령만 표시)
- 모바일 환경에서는 사용자 상호작용 후 오디오 재생 가능
- HTTPS 환경에서 오디오 재생이 더 안정적
